# MediScan UI

A basic static website built with plain HTML, CSS, and JavaScript.

## Project Structure

```
medscan-ui/
├── html/
│   └── index.html      # Main HTML file
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # Main JavaScript file
├── assets/             # Images, icons, and other static assets
└── README.md           # This file
```

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Contact form with validation
- Service cards with hover effects
- Modern CSS with CSS variables for easy customization
- Clean, semantic HTML structure
- Vanilla JavaScript (no dependencies)

## Getting Started

1. Open `html/index.html` in a web browser to view the site
2. Customize the content and styles as needed
3. Add images and assets to the `assets/` folder

## Customization

- Edit `html/index.html` to change the page content
- Modify `css/style.css` to update colors and styling
- Update `js/script.js` to add custom functionality

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design for all screen sizes
