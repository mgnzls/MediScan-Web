# MediScan-Web

A web-based medical scanning application.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [Git](https://git-scm.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/mgnzls/MediScan-Web.git
   cd MediScan-Web
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm start
   ```

### Git Setup (first-time contributors)

If you are pushing a local copy of this project to GitHub for the first time:

```bash
git branch -M main
git remote add origin https://github.com/<YOUR_USERNAME>/MediScan-Web.git
git push -u origin main
```
